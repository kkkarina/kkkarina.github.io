Профессиональные навыки:
  • Знание языков верстки и веб-дизайна HTML и CSS (HTML5, CSS3, flexbox, media-запросы, CSS-анимация).
  • Базовые знания XML.

  • Владение Photoshop (редактирование изображений, планирование и создание PSD-макетов, создание иконок для сайтов).
  • Опыт вёрстки landing page.
  • Знание языка программирования JavaScript (JQuery, ООП, AJAX).
  • Опыт работы с animate.css, wow.js, owlCarousel.
  • Базовые знания Node.js.
  • Знание языка программирования PHP (базовые знания, ООП, опыт написания ядра сайта, работа с семантическим URL, MVC).
  • Знание языка SQL (MySQL и Oracle).
  • Опыт работы с СУБД MySQL (PHPMyAdmin).
  • Умение проектировать реляционные базы данных.
  • Знание языка программирования C++ (базовые знания, ООП).
  • Знание основных шаблонов проектирования в программировании.
  • Опыт работы с Apache, Git, Denwer, CMS WordPress.
  • Знания CMS 1С-Битрикс со стороны Контент-менеджера.
Практический опыт:
  • Верстка сайта по заданному макету (ручная, с использованием Bootstrap 3,4).
  • Опыт написания программы на языке JavaScript , которая предоставляет функционал по разбору html-документа, выделению его составных частей и их замене (например, замена изображений, надписей, фона и т.д.).
  • Опыт создания сайта на CMS Wordpress. Осуществлены дизайн и верстка сайта с помощью HTML5, CSS3, Bootstrap 3, animate.css, wow.js. Настроена CMS Wordpress (с использованием языка PHP). Подключен эквайринг от Сбербанка. На языке JavaScript написан плагин «Онлайн-калькулятор пластиковых окон».
  • Опыт обработки изображений на JavaScript.
  • Опыт написания программы на JavaScript по подгрузке (с помощью Ajax) информации из базы данных и генерации html-кода на странице.
